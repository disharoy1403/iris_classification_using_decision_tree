The Iris dataset contains 150 samples of iris flowers with four features:
- Sepal Length

Each sample belongs to one of three species:
- *Iris-setosa*  
- *Iris-versicolor*  
- *Iris-virginica*  

This project:
- Loads and preprocesses the dataset  
- Encodes class labels into numeric values  
- Trains a Decision Tree model  
- Evaluates performance using accuracy and classification report  
- Visualizes the trained decision tree
